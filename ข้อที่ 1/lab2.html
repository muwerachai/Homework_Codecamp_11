Lab2: การหาข้อมูล Tag

  1. Q: ค้นหาข้อมูลว่าแท็ก มีหน้าที่อะไร?

    2. Ans: บอกให้ browser ว่าเป็น html

  2. Q: ถ้าอยากให้มี tooltip ใน tag นั้นต้องใช้ attribute ใด?

    3. Ans: ใช้ title attribute

  3. Q: แท็ก < b >, < i > ต่างจาก < strong >, < em > หรือไม่ อย่างไร?

    4. Ans: ไม่ต่างกันในเรื่องของการแสดง < b >, < strong > ตัวอักษรจะเป็นตัวหนาเหมือนกัน ส่วน < i >, < em > ตัวอักษรจะเป็นตัวเอียงเหมือนกัน แต่ต่างตรงที่ Browser จะให้ความสำคัญ < strong >, < em > มากกว่า เนื่องจากจะช่วยให้ผู้ใช้ search หาคำนั้น แล้วพบกับเพจที่ตรงกับความต้องการของผู้ใช้สูง

  4. Q: หา tag ที่สามารถแสดงข้อความกลับหลังได้

    5. Ans: < bdo dir = " " > : < /bdo >
